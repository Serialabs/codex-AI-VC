# Site B: 404 Page Taste Reference

**URL:** Any non-existent route  
**Slug:** `404`  
**Purpose:** Display a friendly 404 not found page when users navigate to a non-existent URL.

---

## Taste Signature Rules

### 1. Minimal Design
- Simple page with dramatic "404" heading
- Subheading indicating page was not found
- Maintains brand consistency

### 2. Consistent Aesthetic
- Dotted grid background remains
- Colors consistent with other pages
- Playful but not distracting

---

## Visual Treatment

### Background System
- Same cream dotted grid as other pages
- Clean and uncluttered

### Typography
- "404" in massive display size (~72px or larger)
- Could use script font for playfulness
- Or bold sans-serif for impact
- Subheading in regular weight

### Layout
- Centered content
- Generous whitespace
- Optional illustration or icon

---

## Content

```
404

Oops! This page doesn't exist.

[Back to Home]
```

---

## Key Patterns for AIVC

### Error States
The 404 pattern can be used for:
- Report not found
- Invalid company slug
- General error pages

### Consistent Branding
- Maintain the cream background
- Use script or display typography
- Include navigation back to main content

---
